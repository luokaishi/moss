## 🔍 技术细节

### 涌现检测机制

**输入**: Agent行为历史（最近100个行为）  
**处理**:
1. GoalDiscoverer生成候选目标
2. 新颖性评估（≥0.7）
3. 四维度独立性验证
4. 涌现事件记录

**输出**: 验证通过的涌现目标

### 四维度验证

1. **列表独立性**: 目标不在预定义列表中
2. **语义独立性**: 语义相似度 < 0.7
3. **来源独立性**: 来源行为组合唯一
4. **因果独立性**: 因果置信度 ≥ 0.6
