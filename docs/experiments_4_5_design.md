# MOSS实验4/5设计文档

## 实验4：真实API环境测试

### 目标
验证MOSS在与真实世界约束（成本、配额、失败）交互时的自适应行为

### 设计
- **环境**: APIEnvironment
  - 每日配额限制（1000次/天）
  - 预算约束（$10初始 + $5/日补贴）
  - API调用成本（$0.01/次）
  - 5种API：search, calculate, store, notify, learn
  - 高峰期成功率降低

- **Agent**: RealWorldMOSSAgent
  - 四目标决策（生存/好奇/影响/优化）
  - 知识积累
  - 声誉系统
  - API效果学习

### 初步结果（需调整）
- 运行时长：168小时（1周）
- 观察：Agent过于保守，主要使用calculate API
- 知识获取：0（未充分探索）
- 生存：100%（但无成长）

### 待优化
1. 提高探索奖励系数
2. 降低calculate的效果，迫使尝试其他API
3. 增加环境变化频率，迫使适应

## 实验5：长期演化与涌现智能

### 目标
验证多代演化中的：
1. 基因型向最优策略演化
2. 知识跨代累积
3. 社会结构复杂化
4. 类似"文化"的涌现

### 设计
- **环境**: ComplexEnvironment
  - 100x100资源地图
  - 季节循环（资源波动）
  - 知识池（100个知识单元）
  - 社会网络支持

- **Agent**: EvolvableAgent
  - 可突变基因组（8个基因）
  - 知识传承（父母→子女）
  - 有性/无性繁殖
  - 社会网络构建

- **演化机制**:
  - 100代
  - 每代300步
  - 繁殖条件：资源>0.6, 能量>0.5, 年龄20-200
  - 选择压力：自然死亡 + 适应度评估

### 初步结果（需调整）
- 第0代全部死亡（初始参数问题）
- 资源消耗过快
- 需要调整初始资源量和消耗率

### 待优化
1. 增加初始资源（0.5 → 0.8）
2. 降低能量消耗率（0.01 → 0.005）
3. 调整繁殖条件（降低门槛）
4. 增加资源获取量
