## 📊 数据文件

### 分析脚本

```python
# 语义分析脚本
import re
from pathlib import Path
from itertools import permutations

behaviors = ['adapt', 'communicate', 'coordinate', 'create', 
             'explore', 'help', 'learn', 'optimize', 'protect', 'share']

# 提取目标
goals = set()
pattern = r'发现新颖目标: ([a-z_]+)'
for log_file in Path('oef_real_data').glob('*/experiment.log'):
    with open(log_file) as f:
        goals.update(re.findall(pattern, f.read()))

# 分析覆盖率
theoretical = len(list(permutations(behaviors, 2))) + \
              len(list(permutations(behaviors, 3)))
coverage = len(goals) / theoretical * 100

print(f'总目标: {len(goals)}')
print(f'覆盖率: {coverage:.1f}%')
```

- N=50日志: `oef_real_data/batch_validation_n50/experiment.log`
- N=100日志: `oef_real_data/batch_validation_n100/experiment.log`
