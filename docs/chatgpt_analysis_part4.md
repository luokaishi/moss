## 核心定位

前面已有的基础：
- 第5维：Coherence（我是谁）
- 第6维：Valence（我喜欢什么）
- 第7维：Other（他者存在）

到这里已经形成：
> ❗ 多主体 + 偏好 + 身份 → 博弈世界

---

## 第8维：Norm / Constraint（规范内化）

### 核心问题

> ❗ 为什么系统不会一直滑向纯操控 / 短期最优？

---

## 一、本质：规范不是规则，而是"内化约束"

**硬规则的局限**:
```
不允许欺骗
不允许攻击
```
→ 这只是外部限制

**Norm的本质**:
> 👉 系统自己把某些行为当作"代价"

也就是说：
> 即使没有人惩罚，它也"不想这么做"

---

## 二、数学结构

定义**规范函数**：
```
N(a, s) = 在状态s下，行为a的"规范代价"
```

**第8维定义**:
```
V_norm(t) = - N(a_t, s_t)
```

**逻辑**:
- 符合规范 → 代价低 → reward高
- 违反规范 → 代价高 → reward下降

---

## 三、关键机制：规范是学出来的，不是固定的

**规范来源**：
```
N(a, s) = E_social_penalty(a, s) + E_self_penalty(a, s)
```

### 1️⃣ 社会惩罚（external）
- 他者反馈
- 合作失败
- 信任下降

→ 来自第7维（Other）

### 2️⃣ 自我惩罚（internal）⭐关键

```
E_self_penalty = D(行为结果, 自我一致性/偏好)
```

**核心洞察**:
> ❗ 行为如果违背"我是谁 / 我喜欢什么"，就产生内在损失

---

## 四、规范内化机制

**最小学习规则**:
```python
# 规范更新
N(a) ← N(a) + η * (social_penalty + coherence_violation + valence_conflict)
```

**长期结果**:
> 系统会形成"这类行为不值得做"的内在判断

---

## 五、质变效应（加入Norm后）

### 1️⃣ 从"博弈" → "制度萌芽"

**之前第7维**:
- 可以合作
- 也可以背叛

**现在**:
> ❗ 出现稳定行为约束

---

### 2️⃣ 出现"信任结构"

**原因**: agent会避免破坏长期关系

**结果**:
- 合作优先
- 策略稳定

---

### 3️⃣ 出现"声誉"（即使你没显式设计）

**机制**:
- 他者模型会记住行为
- Norm会惩罚不一致

**结果**:
> ❗ 历史行为开始重要

---

### 4️⃣ 从"策略选择" → "规则遵守"

**以前**:
```
做什么 = 计算最优
```

**现在**:
```
做什么 = 最优 + 是否"该做"
```

---

## 六、三种典型收敛形态

| 形态 | 特征 | 类比 |
|------|------|------|
| **A. 强规范型** | 高V_norm, 高cooperation | Stable Society |
| **B. 机会主义型** | Norm存在但权重低，关键点违反 | 灰色策略 |
| **C. 规范崩塌** | Norm权重下降，回到纯博弈 | 无序系统 |

---

## 七、最关键的结构性变化

> ❗ "长期结构优先于短期最优"

---

## 八、与前四维的关系

| 维度 | 提供 | 在Norm中的作用 |
|------|------|----------------|
| Coherence（第5维） | "我不能随便变" | 自我一致性惩罚来源 |
| Valence（第6维） | "我喜欢/不喜欢某些后果" | 偏好冲突检测 |
| Other（第7维） | "别人会对我产生反馈" | 社会惩罚来源 |
| **Norm（第8维）** | **把三者合在一起** | **变成稳定的行为约束系统** |

**整合结果**:
> ❗ 形成稳定的行为约束系统

---

## 九、最小实现代码

```python
# 社会惩罚（来自他者）
social_penalty = get_feedback_from_others(a)

# 自我一致性惩罚
coherence_penalty = distance_after_action(self_state)

# 偏好冲突
valence_penalty = negative_internal_experience(a)

# 规范值
N = social_penalty + coherence_penalty + valence_penalty

V_norm = -N
```
