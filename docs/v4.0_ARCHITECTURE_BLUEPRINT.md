### 设计目标
1. **World Model**: State → Action → Transition → Prediction
2. **LLM推理层**: Purpose → Language → Reasoning → Action
3. **开放式目标空间**: Agent发明新objective维度
4. **内在代价**: 能量、风险、机会成本
5. **混沌适应性**: 允许崩溃、相变、重组

### 3.4 Cost-Benefit Evaluation (代价评估)

**功能**: 内在代价机制

```python
class CostBenefitEvaluator:
    def __init__(self):
        self.energy_budget = EnergyBudget()
        self.risk_estimator = RiskEstimator()
        self.opportunity_cost_calculator = OpportunityCostCalculator()
    
    def evaluate_action(self, action, predicted_outcome) -> Evaluation:
        """
        全面评估action的代价和收益
        
        Components:
        - Benefit: fulfillment of purpose
        - Cost: energy + risk + opportunity cost
        - Net value: benefit - cost
        """
        energy_cost = self.energy_budget.estimate(action)
        risk = self.risk_estimator.estimate(action, predicted_outcome)
        opportunity_cost = self.opportunity_cost_calculator.calculate(action)
        
        return Evaluation(
            benefit=predicted_outcome.reward,
            cost=energy_cost + risk + opportunity_cost,
            net_value=predicted_outcome.reward - (energy_cost + risk + opportunity_cost),
            uncertainty=predicted_outcome.uncertainty
        )
    
    def update_energy_budget(self, actual_energy_used):
```

**关键特性**:
- 能量不是无限的（约束过度探索）
- 风险估计防止灾难
