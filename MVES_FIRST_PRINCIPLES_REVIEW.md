## 📊 一、定义与范围

### 1.1 核心定义

**MVES (Minimal Viable Evolutionary System)**:
> 一个研究多 Agent 协作系统中自驱力演化机制的**实验性框架**。

**基本组成**:
- 四目标系统 (生存/好奇/影响力/优化)
- 多 Agent 协作机制
- 开放目标生成
- 长期运行能力

### 1.2 范围界定

| 属于 MVES | 不属于 MVES |
|-----------|-------------|
| 多 Agent 协作框架 | 通用 AGI 系统 |
| 自驱力演化研究 | 人类水平智能 |
| 实验性验证平台 | 生产级应用 |
| 学术研究框架 | 商业化产品 |

**结论**: MVES 是**研究工具**，非 AGI 产品。
