### v1: 基础策略演化

**实验设计**:
```python
# 硬编码策略
strategies = ['random', 'conserve', 'explore']

# 能量约束 + 死亡机制
if energy <= 0:
    agent.die()

# 随机变异
if random.random() < mutation_rate:
    agent.strategy = random.choice(strategies)
```

**实验结果**:
| 指标 | 数值 |
|------|------|
| 代数 | 100 |
| 死亡 | 0 |
| 策略跃迁 | 1 次 (random→explore) |
| 最终能量 | -32 |

**关键发现**:
- ✅ 策略可以自发跃迁
- ⚠️ 死亡机制未触发（fitness = energy + steps 设计缺陷）
- ⚠️ 选择压力不足

**结论**: 基础演化机制成立，但需要更强的选择压力。

### v3: Code as Genome

**实验设计**:
```python
# brain.py 作为可演化基因
class Agent:
    def __init__(self):
        self.brain = load_brain('brain.py')
    
    def act(self, state):
        return self.brain.decide(state)

# 自修改代码机制
def mutate(self):
    new_brain = self.brain.copy()
    new_brain.code = self._mutate_code(new_brain.code)
    return new_brain
```

**实验结果**:
| 指标 | 数值 |
|------|------|
| 代数 | 100 |
| 自修改 | 42 次 |
| 代码退化 | 2 agent |
| 平均变异 | 8.4/agent |

**关键发现**:
- ✅ Code as Genome 可行
- ⚠️ 文本替换导致代码退化
- ✅ 回退机制防止崩溃

**结论**: 代码演化成立，但需要更智能的变异机制。

# 三大驱动
