## 3. 关键洞察

### 3.1 Purpose演化的决定因素

**Run 4.x (环境驱动)**:
```
外部Phase变化 → Purpose适配 → 行为调整
```

**72h实验 (任务驱动)**:
```
预设任务列表 → 工具调用 → Purpose标签分配
```

### 3.2 对v5.0设计的启示

**问题**: 72h实验中Purpose演化的"自驱力"体现不足

**原因**:
1. 任务来自预设列表，非自生成
2. 缺乏环境压力梯度
3. Purpose作为标签而非驱动力

**v5.0改进方向**:
1. **开放任务生成**: Agent自主决定做什么
2. **环境反馈闭环**: 行动结果影响后续目标
3. **多层级目标**: 短期任务 vs 长期Purpose动态关联
