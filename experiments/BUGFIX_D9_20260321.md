# 72小时实验 Bug 修复记录

## Bug: D9 (Purpose强度) 为负值

**发现时间**: 2026-03-21 09:47  
**修复时间**: 2026-03-21 09:48  
**影响范围**: Step 0 - Step 20000 (预计)

### 问题描述
Purpose Generator 的第9维 (Purpose强度) 计算为负值：
```
D9 = coherence_score * 0.2
```
由于 `coherence_score` 为负，导致 D9 为负。

### 修复方案
将计算改为取绝对值：
```python
purpose_9th = abs(coherence) * 0.2  # 确保非负
```

### 影响分析
- **已生成的数据**: Step 0-18000 的 D9 值为负（历史数据）
- **修复生效**: Step 20000 起（下次 Purpose 生成）
- **核心功能**: 不受影响，Purpose 演化仍在正常工作

### 数据说明
实验报告中应注明：
- 早期数据 (Step < 20000) 的 D9 值为负，这是已知 Bug
- 修复后的数据 (Step >= 20000) D9 值正常
- 这不影响 S/C/I/O 四个维度的演化分析
