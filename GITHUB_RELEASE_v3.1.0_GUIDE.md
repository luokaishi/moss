## How to Create GitHub Release

### Step 1: Go to GitHub Releases Page
Visit: https://github.com/luokaishi/moss/releases

### Step 2: Click "Draft a new release"

### Step 3: Fill in the Form
- **Choose a tag**: Select `v3.1.0` from dropdown
- **Release title**: 
  ```
  MOSS v3.1.0 - Self-Generated Purpose (9D System)
  ```
- **Description**: Copy content from `GITHUB_RELEASE_NOTES_v3.1.0.md`

### Step 4: Publish
Click "Publish release"
